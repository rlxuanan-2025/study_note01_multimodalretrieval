Filename: Multimodal_Retrieval.zip

* MR_train_imgs.tsv：images for training set（base64 encoding）
* MR_train_queries.jsonl：training queries with relevant product ids
* MR_valid_imgs.tsv：images for validation set（base64 encoding）
* MR_valid_queries.jsonl：validaton queries with relevant product ids
* MR_test_imgs.tsv：images for testing set（base64 encoding）
* MR_test_queries.jsonl：testing queries, prediction file, participant should complete the "item_ids" field for each query, which is a Python list data type. Note: the predicted "item_ids" must contain 10 candidates.
* example_pred.jsonl：submission result demo.
* README.txt: readme.
